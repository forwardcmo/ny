# NewYork.com — Investor Landing

Static site. No build step, no dependencies (fonts load from Google Fonts CDN).

## Files
- `index.html` — landing page (entry point)
- `deal-room.html` — Deal Room "being updated" placeholder
- `assets/` — hero image + investor PDFs

## Deploy on GitHub Pages
1. Create a new repo and push these files to the repo root.
2. Repo Settings → Pages → Source: `main` branch, `/ (root)`.
3. Live at `https://<user>.github.io/<repo>/` within a minute.

For a custom domain (e.g. invest.newyork.com): Settings → Pages → Custom domain, then add the matching DNS record.
